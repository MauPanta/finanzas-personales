📤📥 Guía: Importación y Exportación de Datos

Transfiere tus datos financieros entre computadoras fácilmente

📤EXPORTACIÓN - Crear Backup

Paso a Paso:

  1. Ve a la sección "Historial de Transacciones"
  2. Haz clic en el botón "📥 Exportar"
  3. Se descarga automáticamente: finanzas-backup-2025-08-29.json
  4. Guarda el archivo en un lugar seguro

¿Qué se exporta?

  • ✅ Todas tus transacciones (ingresos y egresos)
  • ✅ Pagos recurrentes programados
  • ✅ Meta de ahorro configurada
  • ✅ Resumen estadístico completo

📥IMPORTACIÓN - Restaurar Datos

Paso a Paso:

  1. Abre la aplicación en la nueva computadora
  2. Haz clic en el botón "📤 Importar"
  3. Selecciona tu archivo finanzas-backup-YYYY-MM-DD.json
  4. Confirma la importación (lee la advertencia)
  5. ¡Listo! Todos tus datos aparecerán instantáneamente

⚠️IMPORTANTE: Advertencias

  • La importación REEMPLAZA todos los datos actuales
  • No se pueden combinar datos - solo reemplazar completamente
  • Exporta primero si tienes datos importantes en la nueva computadora
  • Verifica el archivo antes de importar

💡Consejos Profesionales

📅 Frecuencia de Backup:

  • Diario: Si usas la app intensivamente
  • Semanal: Para uso regular
  • Mensual: Para uso ocasional
  • Antes de cambios importantes: Siempre

🛡️ Seguridad:

  • Mantén múltiples versiones de backup
  • Almacena en ubicaciones diferentes (USB + Cloud)
  • Documenta cuándo y por qué exportaste
  • Prueba la importación periodicamente
